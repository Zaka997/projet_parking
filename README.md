# projet_parking

Une entreprise souhaite mettre en place une application web interne permettant de gérer les réservations de places de parking par ses salariés. Le parking comporte 10 places, dont 3 réservées aux personnes en situation de handicap.

Les salariés doivent pouvoir réserver une place pour une durée définie dans la limite :
    • Durée maximale : 5 jours consécutifs
    • Durée minimale : 1 jour
